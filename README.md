# Modern Map

This is simply, easily, able to limit, similar to pure object

## Getting started

```bash
# Install via npm
npm install modern-map

# Install via yarn (Recommend)
yarn add modern-map
```

## Compatibility

[all browsers that support Proxy](https://caniuse.com/#search=proxy)
